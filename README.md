# Programming Language Paradigms

Programming Language Paradigms Term Projects.

## Projects

### 1. Racket: Prefix Notation Calculator (Functional Programming)
A functional programming project implementing a custom calculator in Racket that supports prefix notation, history references, and interactive/batch modes.
* **Key Features:** Recursive tokenization, custom expression evaluation (REPL), and state management via history recursion.

### 2. Prolog: Work Schedule Planner (Logic Programming, Declarative)
A logic programming project using Prolog to solve constraint satisfaction problems (scheduling employees across shifts and workstations).
* **Key Features:** Constraint-based task assignment, backtracking search, and automated test predicates.

## Tech Stack
* **Languages:** Prolog (SWI-Prolog), Racket (Scheme/Lisp family)
